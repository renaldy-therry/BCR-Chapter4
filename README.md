## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/renaldy-therry/BCR-Chapter4.git
   ```
2. Install packages
   ```sh
   npm install
   ```
3. Next Step

   ```sh
   - setting config.js 
   - npx sequelize db:create
   - npx sequelize db:migrate
   ```

4. Run
   ```sh
   npm run start
   ```
